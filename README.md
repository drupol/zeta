# Plot the Riemann Zeta function

![](./zeta.svg)

An educational project to learn [Octave][octave website] and plot the
[Zeta][zeta wikipedia] function.

## Testing

```shell
nix develop
```

Then run the applet with octave:

```shell
octave src/zeta.m
```
